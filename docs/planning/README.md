## 사용 방법

1. 통합 로드맵에서 분기 목표와 milestone ID를 확인한다.
2. 실행 보드에서 milestone ID 기준으로 Now 항목만 진행한다.
3. 스프린트 백로그에서 작업 단위/완료 기준으로 이슈를 분해한다.
4. 운영 절차는 RUNBOOK 기준으로만 실행한다.
5. 아키텍처 변경은 ADR 먼저 기록한다.
