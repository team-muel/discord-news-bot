# ADR-004: 근거 우선 응답 규격

- 상태: Accepted
- 날짜: 2026-03-12

## 배경

고맥락 에이전트는 답변 자체보다 근거 추적 가능성이 중요하다.

## 결정

모든 에이전트 응답은 다음 형식을 따른다.

1. 결론(요약)
2. 근거(메모리 ID/메시지 ID)
3. 신뢰도(낮음/중간/높음)
4. 다음 행동 제안(선택)

근거가 부족하면 확정적 어조를 금지한다.

## 근거

- 근거 없는 응답은 운영 리스크가 높다.
- 관리자 검증과 디버깅 속도를 높인다.

## 결과

- 응답의 감사 가능성 향상
- 운영 정책 질의에서 오판 리스크 감소
